

### UDI (Undergraduate Development Internship) 이란?
- UDI는 Robotics and Computer Vision Lab에서 운영하는 학부생 개발 인턴십 프로그램으로, 학생들이 실제 AI·로보틱스 연구개발 프로젝트에 참여하여 차세대 지능형 로봇 기술을 개발하는 프로그램입니다.
- URP가 새로운 지식을 발견하고 연구 성과를 창출하는 데 중점을 둔다면, UDI는 최신 AI·로보틱스 기술을 실제 시스템으로 구현하고 제품 수준의 소프트웨어를 개발하는 데 중점을 둡니다

### UDI 지원 전 체크리스트
- 나는 연구자가 아닌 개발자로 성장하기 바라는가
- 나는 세종RCV연구실의 URP와 UDI 프로그램을 구분할 줄 알고, 목적에 맞게 지원하고 있는가
